## 初期化

`foo` の部分は自分のリポジトリ名にしてください。事前に[スポンサー](/sponsors)の登録と [gh](https://cli.github.com/) のインストールが必要です。

### リポジトリを作成する場合

```sh
gh repo create my-app --template nino-plus/turborepo-template --clone --private
```

### リポジトリの作成が不要な場合

クローン後、手動でリモートリポジトリを接続してください。

```sh
git clone https://github.com/nino-plus/turborepo-template.git my-app
```

### 内容

- Turborepo
- Better Auth
- nuqs
- Drizzle
- shadcn/ui
- Resend
- Storybook
- Playwright
- Vitest

などが入ったテンプレートリポジトリをGitHub上に作成し、コマンドを実行したディレクトリにクローンされます。

## セットアップ

```sh
pnpm install
pnpm run setup # 環境変数ファイルやローカルDBファイルの作成
pnpm dev
```
