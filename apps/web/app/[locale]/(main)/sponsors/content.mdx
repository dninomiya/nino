# スポンサー

発信や活動を支えてくれるスポンサーを募集しています。

- X や YouTube の発信が役に立った
- 公開ドキュメントやレジストリアイテムが役に立った

という方はぜひスポンサーとなり、ご支援をお願いします。

## スポンサー特典

- すべてのドキュメントを閲覧できます。
- すべてのソースコードを参照できます。
- すべてのレジストリアイテムを使用できます。
- GitHub組織に参加し、実務レベルのソースコードにアクセスできます。
- Discordのスポンサー限定チャンネルに参加できます。
- 毎週土曜に開催されるオンライン勉強会、交流会に参加できます。
- Discordで技術的な質問、相談が無制限にできます。
