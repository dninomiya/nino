## 基盤

- フレームワーク / [Next.js](https://nextjs.org/)
- 認証 / [Better Auth](https://better-auth.com/)
- ORM / [Drizzle](https://orm.drizzle.team/)

## UI

- 汎用UI / [shadcn/ui](https://ui.shadcn.com/)
- リッチエディター / [Tiptap](https://tiptap.dev/)
- コードハイライト / [Shiki](https://shiki.style/)
- 仮想スクロール / [virtua](https://github.com/inokawa/virtua)
- AIチャット / [Vercel AI SDK](https://ai-sdk.dev/docs/introduction)
- クライアントフェッチ / [SWR](https://swr.vercel.app/)


## コード管理

- ディレクトリ整理 / レイヤードアーキテクチャ
- Lint / [ESLint](https://eslint.org/)（shadcn/ui の対応に合わせて [Biome](https://biomejs.dev/) を検討）
- Format / [Prettier](https://prettier.io/)（shadcn/ui の対応に合わせて [Biome](https://biomejs.dev/) を検討）
- モノリポ / [Turborepo](https://turborepo.com/)

## BaaS

- Storage / [Cloudflare R2](https://www.cloudflare.com/products/r2/)
- Database / [Turso](https://turso.tech/)
- Payment / [Stripe](https://stripe.com/)
- Email / [Resend](https://resend.com/)
- API / [AI Gateway](https://vercel.com/ai-gateway)
- Hosting / [Vercel](https://vercel.com/)
- Cron / [Vercel Cron Jobs](https://vercel.com/docs/cron-jobs)
- Realtime / [Ably](https://ably.com/)
- Video / [Vimeo](https://vimeo.com/)
