## 2025年11月11日 - スポンサー機能追加

- [スポンサー](/sponsors)

スポンサー登録が可能になりました。スポンサーは限定コンテンツにアクセスできます。スポンサー機能は [nino+](https://nino.plus) のサブスクリプションと連動しています。将来的に nino+ は当サイトのスポンサーに統合される予定です。

## 2025年10月20日 - リニューアル

最新の技術スタックに合わせてリニューアルしました。

- Next.js 16: Framework
- Turso: Database
- Drizzle: ORM
- Vercel - Hosting
- shadcn/ui: UI Framework
- Better Auth: Authentication
- Turborepo: Monorepo
- Custom i18n: Internationalization

### 特徴

- クライアントフェッチを活用し、全画面を静的生成にすることで初期表示の高速化を実現しています。
- 国際化はスクラッチしています。React Cache を Proxy(旧Middleware) を活用し、静的生成のままパーソナライズを実現しています。
- MDXでコンテンツマネジメントを行なっています。
