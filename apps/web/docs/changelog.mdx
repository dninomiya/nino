## 2025年10月20日 - リニューアル

最新の技術スタックに合わせてリニューアルしました。

- Next.js 16 - Framework
- Turso - Database
- Drizzle - ORM
- Vercel - Hosting
- shadcn/ui - UI Framework
- Better Auth - Authentication
- Custom i18n - Internationalization

### Next.js 16

- Next.js 16 にアップデートしましたが cacheComponent は有効にしていません。
- クライアントフェッチを活用し、全画面を静的生成にすることで初期表示の高速化を実現しています。

## Turso

- 維持コストが安いため、Turso を採用しました。
- ベクトル検索を活用し、AIフレンドリーな実装を計画しています。

## Drizzle

ORM は Drizzle を採用しました。

## Vercel

- Vercel にデプロイしています。

## shadcn/ui

- shadcn/ui を採用しました。

## Better Auth

- 認証は Better Auth を採用しました。
