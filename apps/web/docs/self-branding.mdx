有効順に

1. 有名企業や有名プロダクトの役職に就く（テックリード、PMなど）
2. **品質が高く、実際に使われるマイプロダクトを公開する**
3. Zenn や X でアウトプットする
    - コメントし、交流する
4. 交流会に参加する

かけだしの場合 1 は無理なので、2 に投資しつつ、その過程で得た知見を 3 でアウトプットしたり、並行して交流会や勉強会で人脈を広げるのが良いです。

## X のコツ

本来のつぶやきツールとして使う場合スルーでOKですが、転職市場を想定してブランディングを行う場合は以下の点に気を付けて発信しましょう。

- 有益な情報のみ発信する
- プライベートな話題は1割以下に抑える
- 自分のコラムや主張は控える
- 自身の関わる組織やプロダクトに関するインサイダー視点の発信（NDAに反しない範囲）
    - たとえばメルカリの中の人が、「メルカリでは主にNext.jsが使われている、などといった発信」

海外の著名エンジニアも9:1の割合で有益な情報やプロダクトのアップデートに徹しています。

## コメントする

とはいえそんなにネタがない場合、自分が志望する業界の人をフォローし、Zenn や リプでさりげなく接点を持って関係や認知を広げていくと良いです。「あ、このアイコンどこかで見たな」という認知を広げるのが大事。
